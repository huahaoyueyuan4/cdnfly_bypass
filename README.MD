# CDNFLY 算法绕过
## 0. 作者的话
开发本项目纯属学习交流，我希望能提升tg黑客的水平，不要再被卖二手脚本的圈，本项目不能直接用于攻击，但是稍微有点开发能力的人都开源与现有的脚本结合。
## 1. 环境准备
Python 3.8+
## 2. 安装相关依赖
```shell
pip install -r requirements.txt
```
## 3. 运行
```shell
python 你要运行的脚本.py
```
## 4. 注意事项
- 如果你要解析旋转图片，请先解压缩img.7z，然后运行`server.py`，然后再运行`rotate.py`
- 压缩文件是分片压缩，如果不会解压建议百度
# 项目结构
```
rotate - 旋转图片
auto - js质询
click - 点击盾
delay_jump - 5秒盾
slide - 滑块
```
